# CodSoft Python Internship Tasks

This repository contains the tasks completed during my Python Programming Internship at CodSoft.

## Tasks Completed

### 1. Calculator
A command-line calculator developed using Python that performs:
- Addition
- Subtraction
- Multiplication
- Division
- Input validation and error handling

### 2. Password Generator
A secure password generator that:
- Generates random passwords
- Supports uppercase letters, lowercase letters, digits, and symbols
- Allows custom password lengths

### 3. Rock Paper Scissors Game
A Python implementation of the classic game featuring:
- User vs Computer gameplay
- Random computer choices
- Score tracking for wins, losses, and ties
- Multiple rounds support

## Technologies Used
- Python 3
- `random` module
- `string` module
- `secrets` module
