# Rock Paper Scissors Game using Python

## Description
This is a command-line implementation of the classic Rock Paper Scissors game built using Python. The user plays against the computer, which randomly selects its move each round.

## Features
- User can choose between Rock, Paper, and Scissors.
- Computer generates random choices using Python's `random` module.
- Displays both user and computer choices.
- Determines the winner according to game rules.
- Keeps track of:
  - User wins
  - Computer wins
  - Ties
- Allows multiple rounds in a single session.
- Displays the final score before exiting.

## Technologies Used
- Python 3
- `random` module

## How to Run
1. Clone the repository or download the source code.
2. Open a terminal in the project directory.
3. Run the following command:

```bash
python rock_paper_scissors.py
```

## Example

```
* R for Rock
* P for Paper
* S for Scissors *

Enter Choice : R

User's Choice     : Rock
Computer's Choice : Scissors

***User Won***
```

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choices result in a tie
